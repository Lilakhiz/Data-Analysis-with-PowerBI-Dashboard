# 🛍️ Customer Shopping Behavior Analysis

> A complete end-to-end data analytics project that transforms raw customer purchase data into actionable business insights using **Python, SQL, MySQL, and Power BI**.

---

## 📌 Project Overview

This project analyzes **3,900 customer shopping records** to understand purchasing behavior, customer demographics, spending patterns, subscription impact, discount usage, and shipping preferences.

The goal is to help businesses make **data-driven decisions** through data cleaning, exploratory data analysis (EDA), SQL queries, and interactive Power BI dashboards.

---

* 🐍 Data Cleaning & Analysis using Python

## 📈 Key Insights

### 💵 Revenue by Gender

* Female customers generated slightly higher total revenue than male customers.

---

### 🎁 Discount Analysis

* High-spending customers frequently used discounts.
* Premium discount campaigns can improve customer retention.

---

### 🚚 Shipping Preferences

* Customers choosing **Express Shipping** spent approximately **12% more** per transaction than Standard Shipping users.

---

### ⭐ Subscription Impact

Subscribers demonstrated:

* Higher average spending
* Greater contribution to overall revenue
* Better customer loyalty

---

### 👥 Customer Segmentation

Customers were categorized into:

* 🟢 Loyal Customers
* 🟡 Returning Customers
* 🔵 New Customers

This segmentation helps businesses build personalized marketing strategies.
